Fungsi Piecewise C(x) - Analisis Biaya Berdasarkan Penggunaan Penyimpanan

Kalkulator Biaya Cloud Storage

Biaya untuk 180 GB: Rp 90.000/bulan
Fungsi C(x):
\[ C(x) = \begin{cases} 50.000, & \text{untuk } 0 \leq x \leq 50 \\ 90.000, & \text{untuk } 50 < x \leq 200 \\ 150.000, & \text{untuk } 200 < x \leq 500 \\ 150.000 + 2.000(x - 500), & \text{untuk } x > 500 \end{cases} \]

Analisis untuk Content Creator (180 GB)

Kebutuhan: 180 GB
Paket Terbaik: Standard
Biaya: Rp 90.000/bulan
*Paket Basic tidak tersedia untuk penggunaan di atas 50 GB
Paket Biaya untuk 180 GB Status
Basic Rp 50.000 Melebihi batas kuota (Harus Upgrade)
Standard Rp 90.000 Optimal
Premium Rp 150.000 Overkill (Paket Berlebihan)

Analisis Kekontinuan

Fungsi C(x) tidak kontinu pada titik:

x = 0 - 50 GB: Harga dimulai pada Rp 50.000
x = 51- 200 GB: Harga melonjak dari Rp 50.000 menjadi Rp 90.000
x = 201 - 500 GB: Harga melonjak dari Rp 90.000 menjadi Rp 150.000

Fungsi C(x) kontinu pada titik:

x > 500 GB: Rp 150.000 + 2.000 ( x - 500)

Implikasi: Customer perlu berhati-hati dalam memperkirakan kebutuhan karena kelebihan sedikit dari batas paket menyebabkan kenaikan biaya yang signifikan.
